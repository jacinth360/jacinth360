#### [Human Muscle scRNA-seq Analysis](https://github.com/jacinth360/human-muscle-scRNAseq.analysis)
End-to-end single-cell RNA-seq analysis pipeline including QC, normalization, dimensionality reduction (PCA, t-SNE), and Leiden clustering for cell-type identification.

**Tech:** Python, Scanpy, Jupyter Notebook

---

#### [Differential Expression Analysis (WT vs KO)](https://github.com/jacinth360/human-wt-ko-rnaseq-de)
Complete RNA-seq differential expression workflow comparing wild-type and knockout samples, including quality control, alignment, DEA with DESeq2, and functional enrichment analysis.

**Tech:** R, DESeq2, ggplot2, HTML reports

**Tech:** R, Shiny, DESeq2, ggplot2
